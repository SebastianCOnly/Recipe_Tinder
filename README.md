# Recipe Tinder

Recipe Tinder is a simple, mobile-style recipe discovery application inspired by Tinder's swipe-based interaction. Users can browse recipes by swiping through cards and quickly decide whether to like or skip a recipe.

The idea behind this project is to reduce decision fatigue when selecting meals for individuals with family responsibilities and other daily commitments. This app helps to search for recipes online, which can be overwhelming, as many websites are cluttered with excessive text, advertisements, or content locked behind paywalls. Additionally, recipes are often not presented in an engaging or user-friendly manner, making it difficult for users to feel motivated or inspired to cook. There is a clear need for a simple, intuitive, and interactive meal-planning solution that reduces cognitive load while delivering personalized and appealing dish recommendations

The goal of this project is to demonstrate a modern UI interaction pattern for quickly and intuitively discovering cooking recipes. And reducing complexity can improve user engagement, efficiency, and satisfaction in everyday planning applications. 

## Features:

*Third-party recipe APIs for real-time dish retrieval

*Swipe left or right to browse recipes

*View recipe details

*Quick recipe discovery

*Card-based user interface

*Lightweight and simple design
